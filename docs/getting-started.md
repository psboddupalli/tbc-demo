# Getting Started

## Accessing the Sandbox

1. Go to **https://sandbox.saicloud.org**
2. You'll be prompted to log in through our authentication gate — use the email address that was set up for you
3. After authenticating, you'll see the TBC website

## Logging into the Admin Panel

1. Go to **https://sandbox.saicloud.org/admin**
2. Enter your Craft CMS username and password (provided separately)
3. You'll land on the Craft CMS dashboard

## Navigating the Admin Panel

The left sidebar is your main navigation:

- **Dashboard** — Overview and recent activity
- **Entries** — Where all site content lives (pages, news, events, fellows, sponsors)
- **Assets** — Uploaded images, documents, and media files
- **Users** — Your account settings

## What You Can Do

- Edit any page's headline, body text, images, and CTAs
- Add, edit, and remove news articles, events, sponsors, and fellow profiles
- Upload images and documents
- Preview changes before publishing
- Reorder content blocks on pages that use the page builder

## What You Can't Do (By Design)

The sandbox is configured so you can explore content editing freely without risk:

- You **cannot** change the site structure, fields, or content model
- You **cannot** install or remove plugins
- You **cannot** modify system settings or CMS configuration
- You **cannot** delete sections or navigation

This is intentional — it means you can experiment freely without breaking anything structural.

## 5-Minute Walkthrough

Try these to get a feel for the editing experience:

1. **Edit the Invest page** — Go to Entries → Invest in TBC → change the hero headline → Save → refresh the Invest page to see your change live
2. **Add a news article** — Go to Entries → News → New Entry → fill in a headline, excerpt, and body → Save → check the News page
3. **Browse sponsors** — Go to Entries → Sponsors → see all current sponsors with logos. Add a new one and it automatically appears on the Invest page and Homepage
4. **Check events** — Go to Entries → Events → see the seeded events. Each one has date, location, registration URL, description, and event type
5. **Preview on mobile** — Pull up the site on your phone. Everything is responsive

## If Something Goes Wrong

Don't worry — this is a sandbox. Nothing you do here affects the production site. If anything looks broken or confusing, contact Sai at consulting@saicloud.org. The sandbox can be fully reset at any time.
