# Frequently Asked Questions

## General

### What is this sandbox?

This is a private copy of the TBC website proof of concept running on Craft CMS. It's a safe environment for you to explore the content management system, try editing pages, and see what a professional CMS can do for TBC. Nothing you do here affects any live or production systems.

### Who can access this?

Only TBC team members with credentials. The site is behind an authentication gate — you must log in before you can see anything.

### Can I share the URL with others?

Please check with Sai before sharing access. Additional users can be set up quickly if needed.

## Editing Content

### I made a change but I don't see it on the site

- Make sure you clicked **Save** (not just edited the field)
- Try a hard refresh in your browser (Ctrl+Shift+R or Cmd+Shift+R)
- If you're editing a page with a page builder, make sure the block you edited is not collapsed

### I accidentally deleted something

Don't panic. Craft keeps revision history for entries. Open the entry and click the revision dropdown at the top to restore a previous version. If you deleted an entire entry, contact Sai — the sandbox can be fully restored from backup.

### I can't find where to edit something

All content is under **Entries** in the left sidebar. Click on the section name to see entries in that section. If you're looking for a specific page:

- Homepage → Entries → Home
- About pages → Entries → About
- Events → Entries → Events
- News articles → Entries → News & Insights
- Sponsors → Entries → Sponsors
- Fellows → Entries → Fellows

### I see "Changes to these settings aren't permitted"

This is by design. The sandbox is configured so you can edit content but not the underlying site structure (fields, sections, plugins, settings). This prevents accidental structural changes.

### Can I upload images?

Yes. When you see an Assets field, click "Add an asset" to upload or select an existing image. Supported formats: JPG, PNG, WebP, GIF, PDF, and MP4.

## Events

### How do I add a new event?

Go to Entries → Events → + New Entry. Fill in the event name, date, location, type, description, and registration URL. Save, and it will automatically appear on the Events page and calendar.

### How does the calendar work?

The Events page includes a calendar view that automatically shows all events based on their dates. No separate calendar setup is needed — just set the event date correctly and it appears.

### What's a "Flagship" event?

Toggling the Flagship switch on an event makes it appear in a prominent featured band at the top of the Events listing page. Use this for major events like the Capital Investors Summit.
